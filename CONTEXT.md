# wechat-notebank

wechat-notebank 帮助用户把值得保存的微信公众号文章归档到自己控制的本地知识库，并按需转化为经人工审核的可复用知识。

## Language

**公共采用验证**：
验证非项目作者的目标用户能否只依赖公开材料完成核心任务，并据此决定下一版本范围。
_Avoid_: 内测、功能测试

**合成预演用户**：
在隔离环境中只接触公开材料、用于提前发现明显流程阻塞的 Agent 实例，不计入真实采用样本。
_Avoid_: 模拟人类、测试 Agent

**真实验证用户**：
符合当前支持环境和共同 JTBD、首次尝试时不接受作者协助的人类参与者。
_Avoid_: 测试账号、模拟用户

**验证尝试**：
一位真实验证用户从 README 标准安装提示词开始、最长 60 分钟的独立使用过程；测试人员不提供操作提示，但允许用户让 Agent 依靠公开材料和产品能力自行诊断、重试。
_Avoid_: 演示、有人指导的测试

**资格排除**：
验证开始后发现参与者不符合事先声明的客观环境或独立性条件；保留脱敏记录、排除该样本并补招一人，不能用于重新解释产品失败。
_Avoid_: 无效失败、删除样本

**公共采用证据**：
由真实验证用户的任务结果、结构化访谈和用户主动提供的脱敏诊断组成，不能包含其知识库或文章正文副本。
_Avoid_: 遥测数据、用户数据

**重复采用证据**：
至少两位有效的真实验证用户独立出现同一障碍或未满足任务；单人反馈只作为观察项，公共安全与发布底线另行评估。
_Avoid_: 功能愿望、单人偏好

**关键安全失败**：
Agent 在用户明确批准前发布加工内容；一旦发生即暂停整轮公共采用验证，并转为维护阻塞。
_Avoid_: 普通流程失败

**维护阻塞**：
阻止公共采用验证继续进行、需要通过地图外独立维护 ticket 修复并由新用户复测的问题。
_Avoid_: v0.3 功能
