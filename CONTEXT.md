# wechat-notebank

wechat-notebank 帮助用户把值得保存的微信公众号文章归档到自己控制的本地知识库，并按需转化为经人工审核的可复用知识。

## Language

**公共采用验证**：
验证非项目作者的目标用户能否只依赖公开材料完成核心任务，并据此决定下一版本范围。
_Avoid_: 内测、功能测试

**合成预演用户**：
在隔离环境中只接触公开材料、用于提前发现明显流程阻塞的 Agent 实例，不计入真实采用样本。
_Avoid_: 模拟人类、测试 Agent

**真实验证用户**：
符合当前支持环境和共同 JTBD、首次尝试时不接受作者协助的人类参与者。
_Avoid_: 测试账号、模拟用户

**公共采用证据**：
由真实验证用户的任务结果、结构化访谈和用户主动提供的脱敏诊断组成，不能包含其知识库或文章正文副本。
_Avoid_: 遥测数据、用户数据

**维护阻塞**：
阻止公共采用验证继续进行、需要通过地图外独立维护 ticket 修复并由新用户复测的问题。
_Avoid_: v0.3 功能
